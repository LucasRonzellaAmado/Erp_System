# NexusFlow ERP

NexusFlow is an Enterprise Resource Planning (ERP) system developed in PHP designed to manage and organize business operations such as customers, products, inventory, sales, quotes, and suppliers.

The system centralizes important business processes in a single platform, improving organization, efficiency, and operational control.

This project was developed as a personal project and will also be used as a Final Course Project (TCC) for academic purposes.

---

## Features

The system includes several modules for business management:

- Customer registration and management
- Supplier registration and management
- Product registration
- Inventory control
- Sales management
- Quote / budget system
- Cash flow control
- Sales history
- Quote history
- User authentication
- Role-based access control

## Installation

To run this project locally you will need:

- Apache Server
- MySQL Database
- PHP
- WAMP / XAMPP / LAMP environment

### Installation Steps

1. Clone the repository

```
git clone https://github.com/LucasRonzellaAmado/Erp_System.git
```

2. Move the project folder to your server directory.

Example using WAMP:

```
C:\wamp64\www\
```

3. Import the database into MySQL.

4. Configure the database connection inside the project files.

5. Access the system through your browser:

```
http://localhost/Sistema_erp
```
